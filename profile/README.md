# COOP Robotics

SSU Robotics ABC는 협동로봇 기반 무인 편의점 시나리오를 해결하는 프로젝트입니다.

본 프로젝트는 ROS 2, 비전 인식, 음성/VLM 인터페이스, 로봇 매니퓰레이션, 백엔드 서버를 통합하여 두산 M0609와 OnRobot RG2 기반의 pick-and-place 시스템을 구현합니다.
